# Insurance Risk Analytics

A professional machine learning project for insurance risk analytics, covering pricing models, fraud detection, explainability (SHAP), business simulation, and a Streamlit dashboard.

## Project Components
- Insurance pricing models (frequency, severity, pure premium)
- Fraud detection
- SHAP-based model explainability
- Business simulation workflows
- Streamlit dashboard for interactive insights

## Structure Overview
- `data/`: raw, processed, and external data
- `notebooks/`: step-by-step analysis and modeling notebooks
- `src/`: reusable pipelines and modeling code
- `dashboard/`: Streamlit app
- `models/`: trained model artifacts
- `reports/`: figures and final report
- `config/`: project configuration

## Getting Started
1. Create a virtual environment
2. Install dependencies: `pip install -r requirements.txt`
3. Run the main pipeline: `python main.py`
4. Launch the dashboard: `streamlit run dashboard/app.py`
